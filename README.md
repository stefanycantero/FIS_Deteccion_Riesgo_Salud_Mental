# Proyecto de Sistema Experto y Sistema de Inferencia Difusa

## Descripción

Este proyecto es un trabajo en equipo para la asignatura Introducción a la Inteligencia Artificial en la Universidad Nacional de Colombia. Se enfoca en el desarrollo de dos sistemas principales:

1. **Sistema de Inferencia Difusa (FIS)**: Implementado en Java con bibliotecas de lógica difusa.
2. **Sistema Experto (SE)**: Desarrollado en Prolog para recomendaciones personalizadas.

El objetivo es utilizar la inteligencia artificial como una herramienta de alerta temprana en salud mental, proporcionando una evaluación preliminar del riesgo para problemas como depresión, ansiedad e insomnio. 

## Función Principal del FIS

El Sistema de Inferencia Difusa (FIS) tiene como propósito utilizar la inteligencia artificial para servir como herramienta valiosa de alertas tempranas en salud mental. La lógica difusa modela las relaciones entre variables de entrada (como el estado de ánimo, la energía, la calidad del sueño, etc.) para evaluar los niveles de riesgo de depresión, ansiedad e insomnio.

### Proceso del FIS

- **Modelado de Relaciones**: Se utilizan funciones y lógica difusa para establecer cómo las variables de entrada afectan el riesgo.
- **Evaluación de Riesgos**: El FIS asigna conjuntos específicos de valores lingüísticos para las entradas y proporciona una evaluación lingüística del riesgo.
- **Informes para Profesionales**: Los resultados del FIS permiten a los profesionales de salud mental tomar decisiones informadas y diseñar intervenciones personalizadas.

## Función Principal del SE

El Sistema Experto (SE) se basa en los niveles de riesgo proporcionados por el FIS y ofrece recomendaciones desde dos perspectivas:

1. **Autoayuda**: Sugerencias de estrategias, actividades y hábitos para promover la prevención temprana, mejorar el bienestar emocional y reducir factores de riesgo.
2. **Ayuda Profesional**: Recomendaciones sobre la urgencia de consultar a un profesional para iniciar un tratamiento adecuado.

## Utilización de Lógica Difusa

La lógica difusa en el FIS permite:

- **Interpretación de Resultados**: Los resultados se presentan en grados de pertenencia, facilitando la interpretación de los riesgos y la toma de decisiones.
- **Personalización de Evaluación**: La evaluación se ajusta a las circunstancias individuales de cada estudiante, permitiendo una detección temprana efectiva.
- **Adaptabilidad**: El sistema se adapta a las diferencias y factores de riesgo únicos de cada estudiante, mejorando la detección de enfermedades mentales.

## Tecnologías Utilizadas

- **Java**: Para la implementación del Sistema de Inferencia Difusa con librerías de lógica difusa.
- **Prolog**: Para el desarrollo del Sistema Experto en la generación de recomendaciones.

## Contribuciones

Este proyecto es una colaboración entre estudiantes de la Universidad Nacional de Colombia, enfocada en aplicar conceptos de inteligencia artificial para mejorar la salud mental.
